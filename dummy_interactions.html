0/1 Dummy Variable

D = 1 if observation is in group A, 0 otherwise
A dummy (indicator) variable takes the value 1 for one category and 0 for the baseline category.
Use: Gender, union membership, region, time period, policy treated vs control.
Example: Female = 1 if worker is female, 0 if male.

Example: Region: East is baseline, West and Central as dummies.

Including an intercept and dummies for every category causes perfect multicollinearity (dummy variable trap).
Rule: With an intercept, drop one category as the baseline.
Example: For 3 regions, include only 2 region dummies with an intercept.
